# AluminiumPro

A modern aluminium hardware and fittings management platform featuring premium UI, responsive design, inventory organization, and scalable architecture.

## Features
* Premium UI/UX: Designed for a professional and seamless user experience.
* * Inventory Management: Efficiently organize and track aluminium hardware and fittings.
  * * Responsive Design: Fully functional across all devices and screen sizes.
    * * Scalable Architecture: Built with modern technologies to support growth and high performance.
     
      * ## Tech Stack
      * * Frontend: React.js, Tailwind CSS
        * * Backend: Node.js, Express.js
          * * Database: MongoDB / PostgreSQL
            * * Deployment: Vercel
             
              * ## Live Demo
              * Check out the live project here: https://aluminium-pro.vercel.app
